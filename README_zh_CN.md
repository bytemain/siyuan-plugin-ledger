# 思源记账

基于[思源笔记](https://b3log.org/siyuan)的**纯文本复式记账**插件——在笔记中随手记账，数据永远属于你。

![preview](preview.png)

## ✨ 功能特性

- **快速记账** — 通过弹窗或一行速记录入支出、收入和转账（如 `午饭 58`、`2024-03-15 海底捞 258 信用卡`）
- **复式记账** — 每笔交易自动生成借贷平衡的过账分录，遵循标准会计原则
- **斜杠命令** — 在任意文档中输入 `/记账`、`/收入`、`/转账` 或 `/快记` 即可打开记账弹窗
- **快捷键** — `⇧⌘E` 记支出、`⇧⌘I` 记收入、`⇧⌘L` 打开账本总览
- **账本总览** — 专属面板，展示月度收支、结余、支出分类饼图、月度趋势柱状图和最近交易
- **侧栏面板** — 概览面板，一览资产/负债余额和资产净值
- **账户管理** — 按类型（资产、负债、收入、支出、权益）添加、编辑、关闭或重新开启账户
- **导入/导出** — 支持 **Ledger CLI**、**Beancount** 和 **CSV** 格式的导入导出
- **拆分账单** — 一笔交易可拆分到多个支出分类
- **多币种** — 支持 CNY、USD、EUR、GBP、JPY、HKD，货币符号可配置
- **数据自主** — 所有数据以思源块属性形式存储，不会离开你的笔记本
- **国际化** — 完整支持简体中文和英文

## 📦 安装

1. 打开思源笔记 → **集市** → **插件**
2. 搜索 **思源记账**，点击 **安装**

或者从 [Releases](https://github.com/bytemain/siyuan-plugin-ledger/releases) 下载最新的 `package.zip`，解压到 `{工作空间}/data/plugins/siyuan-plugin-ledger/`。

## 🚀 快速上手

1. 在思源笔记中打开任意文档
2. 点击顶栏 💰 图标，或使用快捷键 `⇧⌘E`
3. 填写收款方、金额和账户，点击 **记账**
4. 使用 `⇧⌘L` 打开账本总览，查看财务汇总

### 一行速记

使用快速记账模式，一行录入交易：

```
午饭 58
打车 32 微信
2024-03-15 海底捞 258 信用卡
```

## ⚙️ 设置

打开插件 **设置** 可配置以下选项：

| 选项 | 说明 |
|------|------|
| 默认货币 | 新交易使用的货币（默认：CNY） |
| 默认付款账户 | 未指定账户时使用的默认账户 |
| 显示模式 | 交易块的详细或简洁显示 |
| 自动平衡 | 自动计算最后一个过账行的金额 |

## 🔗 兼容性

- 思源笔记 ≥ 3.0.0
- 支持平台：Windows、macOS、Linux、iOS、Android、HarmonyOS、Docker、浏览器
