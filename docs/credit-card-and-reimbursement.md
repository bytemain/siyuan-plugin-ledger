# 信用卡与报销记账指南

本文档介绍如何在思源记账插件中记录 **信用卡消费** 和 **费用报销** 两种常见场景。

---

## 一、信用卡消费

### 1.1 基本原理

在复式记账中，信用卡属于 **负债（Liabilities）** 账户。刷信用卡消费时，资金流向为：

```
Expenses:消费分类  +金额   ← 消费增加
Liabilities:CreditCard:XX  -金额   ← 负债增加（欠银行的钱）
```

### 1.2 使用弹窗记账

1. 使用快捷键 `⇧⌘E` 或输入 `/记账` 打开支出录入弹窗。
2. **付款账户** 选择对应的信用卡，如 `Liabilities:CreditCard:CMB（招行信用卡）`。
3. **支出分类** 选择消费分类，如 `Expenses:Food:Dining`。
4. 填写金额和收款方，点击记账即可。

### 1.3 使用快记模式

在快记输入框中，可以使用以下简写：

```
海底捞 258 招行信用卡
海底捞 258 信用卡
海底捞 258 工行信用卡
```

支持的信用卡别名：
| 别名 | 对应账户 |
|------|---------|
| `信用卡` | Liabilities:CreditCard:CMB |
| `credit` | Liabilities:CreditCard:CMB |
| `招行信用卡` / `招行` | Liabilities:CreditCard:CMB |
| `工行信用卡` / `工行` | Liabilities:CreditCard:ICBC |
| `花呗` | Liabilities:HuaBei |
| `借呗` | Liabilities:JieBei |

### 1.4 还信用卡（还款）

还信用卡本质上是一笔 **转账**——从资产账户转到信用卡负债账户：

```
Liabilities:CreditCard:CMB  +还款金额   ← 负债减少
Assets:Bank:Checking         -还款金额   ← 资产减少
```

**方式一：使用转账弹窗**

1. 使用 `/转账` 打开转账弹窗。
2. **转出账户** 选择资产账户（如银行卡）。
3. **转入账户** 选择信用卡账户。

**方式二：使用快记模式**

```
还信用卡 5000 银行卡
还招行 5000
还工行 3000 银行卡
```

快记模式会自动识别 `还信用卡`、`还招行`、`还工行` 等关键词，生成正确的转账分录。

## 二、费用报销

### 2.1 场景说明

报销通常分为两步：
1. **先垫付**：自己先付费，记为正常支出，打上 `待报销` 标签。
2. **收到报销**：公司打款后，记为一笔收入，来源为 `Income:Reimbursement`。

### 2.2 步骤一：记录垫付支出

使用正常的支出记录流程，但在 **标签** 中输入 `待报销`：

**弹窗方式：**
1. 打开支出录入弹窗（`⇧⌘E` 或 `/记账`）。
2. 填写金额、收款方等信息。
3. 在 **标签** 栏输入 `待报销`。

**快记方式：**
```
打车去客户现场 86 微信 标签:待报销
2024-03-15 差旅住宿 380 信用卡 标签:待报销
```

### 2.3 步骤二：收到报销款

当公司完成报销后，记录一笔收入：

**弹窗方式：**
1. 使用 `/收入` 或 `⇧⌘I` 打开收入录入弹窗。
2. **收入来源** 选择 `Income:Reimbursement（报销）`。
3. **转入账户** 选择收款的资产账户（如银行卡）。
4. 填写金额，标签可输入 `报销`。

**快记方式：**
```
报销 差旅费 380
报销 打车费 86 银行卡
```

快记模式会自动识别 `报销` 开头的输入，生成收入分录：
```
Assets:Bank:Checking  +380   ← 报销到账
Income:Reimbursement  -380   ← 收入来源
```

## 三、仪表盘与统计

- 信用卡余额可在 **侧边栏** 的负债区域实时查看。
- 报销收入会计入 **收入统计**，在月度收支趋势图中体现。
- 使用 `待报销` 标签可方便地追踪未报销的垫付款项。

## 四、常见问题

**Q: 信用卡消费算不算当月支出？**
A: 是的。在复式记账体系中，消费发生时即记录支出，不论使用何种支付方式。信用卡只是改变了资金来源（从资产变为负债），支出金额不变。

**Q: 报销到账后，这笔消费还算支出吗？**
A: 垫付时记录为支出，报销到账时记录为收入。两者不会互相抵消，在月报中可以分别看到支出和收入的明细。如果需要看"净支出"，可以参考仪表盘中的"本月结余"数据。

**Q: 如何查看所有待报销的支出？**
A: 在交易记录中通过 `待报销` 标签筛选即可找到所有尚未报销的垫付。
