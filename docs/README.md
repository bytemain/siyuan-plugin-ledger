## 快捷键速查

| 操作 | Mac | Windows/Linux |
|------|-----|---------------|
| 记支出 | `⇧⌘E` | `⇧Ctrl+E` |
| 记收入 | `⇧⌘I` | `⇧Ctrl+I` |
| 记转账 | `⇧⌘T` | `⇧Ctrl+T` |
| 打开仪表盘 | `⇧⌘L` | `⇧Ctrl+L` |
| 打开侧边栏 | `⌥⌘L` | `Alt+Ctrl+L` |

| `/转账` `/t` | 转账弹窗 |
