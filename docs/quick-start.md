# 五分钟快速上手

本指南帮助你在 5 分钟内完成插件配置并记录第一笔账目。

---

## 第一步：安装与配置（1 分钟）

1. 在思源笔记的 **集市 → 插件** 中搜索"记账"，找到本插件并安装。
2. 安装后在 **设置 → 插件 → 记账插件** 中完成基本配置：

| 设置项 | 说明 | 推荐值 |
|-------|------|-------|
| 默认币种 | 记账时自动使用的货币 | `CNY`（人民币） |
| 默认支付账户 | 不指定账户时的默认扣款方 | `Assets:Alipay`（支付宝） |
| 显示模式 | 账目在笔记中的展示样式 | `详细模式` |
| 自动平账 | 只输入一边金额时自动补齐 | `开启` |

## 第二步：认识你的账户体系（1 分钟）

插件预置了 30+ 常用账户，覆盖大多数人的日常需求：

```
📂 资产 (Assets)
  ├── 💳 银行卡 (Bank:Checking)
  ├── 🏦 储蓄卡 (Bank:Savings)
  ├── 📱 支付宝 (Alipay)
  ├── 💬 微信 (WeChatPay)
  ├── 💵 现金 (Cash)
  └── 📈 投资 (Investments)

📂 负债 (Liabilities)
  ├── 💳 工行信用卡 (CreditCard:ICBC)
  ├── 💳 招行信用卡 (CreditCard:CMB)
  ├── 🔴 花呗 (HuaBei)
  └── 🟡 借呗 (JieBei)

📂 收入 (Income)
  ├── 💰 工资 (Salary)
  ├── 🎁 奖金 (Bonus)
  └── 📋 报销 (Reimbursement)

📂 支出 (Expenses)
  ├── 🍽️ 餐饮 (Food:Dining)

```

> 💡 这些账户开箱即用，你可以之后随时在账户管理中增删改。详见 [账户体系配置](./account-setup.md)。

## 第三步：记录第一笔支出（1 分钟）

### 方式一：快记模式（最快）

在任意笔记页面输入斜杠命令 `/快速记账` 或 `/q`，在弹出的输入框中输入：

```
午饭 25
```

按回车，完成！插件会自动生成一笔支出记录：
- 支出分类：`Expenses:Food:Dining`（根据"午饭"自动推断）
- 金额：25 CNY
- 付款账户：你设置的默认支付账户

### 方式二：弹窗表单（更详细）

按快捷键 `⇧⌘E`（Mac）或 `⇧Ctrl+E`（Windows），打开支出录入弹窗：

1. 输入**金额**：`25`
2. 输入**收款方**：`午饭`
3. 选择**分类**和**付款账户**（有默认值）
4. 点击**记账**

## 第四步：查看你的账目（1 分钟）

### 仪表盘

按 `⇧⌘L`（Mac）或 `⇧Ctrl+L`（Windows）打开数据仪表盘，你可以看到：

- 📊 本月收支总览（收入、支出、结余）
- 📈 近 12 个月收支趋势图
- 🥧 支出分类饼图（Top 6 分类占比）
- 📋 最近交易记录

### 侧边栏

按 `⌥⌘L`（Mac）或 `Alt+Ctrl+L`（Windows）打开常驻侧边栏，随时查看：

- 各资产账户余额
- 负债账户欠款
- 本月收入/支出
- 净资产

## 第五步：试试更多记法（1 分钟）

快记模式支持灵活的输入格式：

```
# 指定支付方式
咖啡 35 微信
打车 32 支付宝

# 指定日期
2024-03-15 午饭 58

# 添加标签
打车去客户 86 微信 标签:待报销

# 记录收入（使用 /收入 或 ⇧⌘I）
# 记录转账（使用 /转账 或 ⇧⌘T）
```
