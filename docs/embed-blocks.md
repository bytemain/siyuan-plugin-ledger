# 在笔记中嵌入账单

本文档介绍如何在思源笔记的页面中嵌入交易查询，打造个性化的账单视图。

---

## 一、什么是嵌入账单

嵌入账单是一种特殊的思源块，可以在你的笔记页面中直接显示交易记录。它支持多种查询方式，让你不需要打开仪表盘也能在日常笔记中看到相关的账目信息。

典型用途：
- 在日记中查看当天的消费记录
- 在月度总结中嵌入本月账单
- 在项目笔记中追踪项目相关开销
- 创建个人财务概览页面

---

## 二、创建嵌入块

通过斜杠命令创建不同类型的嵌入块：

### 2.1 嵌入本月交易

```
/嵌入本月
```

在当前位置插入一个显示**本月所有交易**的嵌入块。每月初自动显示新月份的数据。

### 2.2 嵌入最近交易

```
/嵌入最近
```

插入显示**最近 20 笔交易**的嵌入块，不限月份。

### 2.3 嵌入所有交易

```
/嵌入所有
```

插入显示**全部交易记录**的嵌入块（分页显示）。

### 2.4 按分类筛选

```
/嵌入分类交易
```

插入按**账户/分类**筛选的嵌入块。可以指定某个分类，如只显示 `Expenses:Food` 下的所有餐饮支出。

### 2.5 按收款方筛选

```
/嵌入收款方交易
```

插入按**收款方名称**筛选的嵌入块。比如查看所有在"海底捞"的消费记录。

---

## 三、使用场景

### 场景 1：日记中嵌入每日开销

在你的每日笔记模板中添加 `/嵌入本月`，就能在日记页面随时看到当月的消费情况。

### 场景 2：月度总结

创建一个"2024年3月财务总结"的笔记：

```markdown
# 2024年3月财务总结

## 本月账单
（这里插入 /嵌入本月）

## 餐饮支出明细
（这里插入 /嵌入分类交易，选择 Expenses:Food）

## 分析与反思
- 本月餐饮支出偏高，主要是因为...
- 下个月计划控制在...
```

### 场景 3：追踪项目开销

如果你在做装修、旅行等项目，可以：

1. 给相关支出打上统一标签（如 `装修`）。
2. 在项目笔记页面使用 `/嵌入分类交易` 查看相关交易。

### 场景 4：财务仪表板页面

创建一个专门的"财务中心"笔记页面：

```markdown
# 🏦 我的财务中心

## 最近交易
（/嵌入最近）

## 本月餐饮
（/嵌入分类交易 → Expenses:Food）

## 本月交通
（/嵌入分类交易 → Expenses:Transport）
```

---

## 四、显示模式

嵌入块会继承你在插件设置中选择的**显示模式**：

### 详细模式

显示完整的交易信息，包括所有过账行和标签：

```
2024-03-15 ✓ 海底捞
  Expenses:Food:Dining    ¥258.00
  Assets:Alipay          -¥258.00
  #聚餐
```

### 紧凑模式

单行显示，更加精简：

```
2024-03-15 ✓ 海底捞  ¥258.00  Food:Dining ← Alipay
```

在插件设置中切换显示模式后，所有嵌入块会自动更新样式。

---

## 五、与交易的交互

嵌入块中显示的交易不只是展示，还可以直接交互：

- **双击**某笔交易：打开编辑弹窗
- **右键**某笔交易：显示编辑/删除菜单
