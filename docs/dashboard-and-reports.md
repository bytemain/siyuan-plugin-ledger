# 仪表盘与数据分析

本文档介绍如何使用仪表盘和侧边栏来分析你的财务数据。

---

## 一、打开仪表盘

三种方式：

1. **快捷键：** `⇧⌘L`（Mac）或 `⇧Ctrl+L`（Windows/Linux）
2. **斜杠命令：** 在笔记中输入 `/仪表盘`
3. **状态栏：** 点击底部状态栏的"本月支出: ¥X,XXX.XX"

---

## 二、仪表盘功能详解

### 2.1 月度概览

仪表盘顶部显示当月（或选中月份）的三个核心指标：

| 指标 | 含义 |
|------|------|
| 💰 总收入 | 当月所有收入的总和 |
| 💸 总支出 | 当月所有支出的总和 |
| 📊 结余 | 收入 - 支出，正数表示当月有盈余 |

### 2.2 月度收支趋势图

以 **12 个月柱状图** 展示近一年的收支变化：

- 🟢 绿色柱子：月度收入
- 🔴 红色柱子：月度支出
- 通过对比可以直观看出：
  - 哪个月花超了
  - 收入是否稳定
  - 整体的收支趋势

### 2.3 支出分类饼图

展示当月 **Top 6 支出分类** 的占比：

- 帮助你了解钱花在了哪里
- 快速识别最大的开销类别
- 每个扇形显示分类名称和百分比

### 2.4 收入来源饼图

展示当月收入来源的构成，了解收入结构。

### 2.5 最近交易列表

显示最近 **20 笔交易** 的明细：

- 日期、收款方、金额、分类
- 点击某笔交易可以快速编辑

### 2.6 月份选择

使用仪表盘顶部的**月份选择器**切换查看不同月份的数据。

## 三、侧边栏（Dock）

### 3.1 打开方式

- **快捷键：** `⌥⌘L`（Mac）或 `Alt+Ctrl+L`（Windows/Linux）
- 打开后，侧边栏会常驻在思源笔记的左侧

### 3.2 显示内容

侧边栏提供实时的财务概览：

```
┌─────────────────────┐
│ 📊 记账插件          │
├─────────────────────┤
│ 资产                 │
│  💳 银行卡    ¥15,000│
│  📱 支付宝    ¥3,200 │
│  💬 微信      ¥1,800 │
│  💵 现金        ¥500 │
├─────────────────────┤
│ 负债                 │
│  💳 招行信用卡 -¥2,300│
├─────────────────────┤
│ 本月收入    ¥15,000  │
│ 本月支出     ¥8,500  │
│ 净资产      ¥18,200  │
├─────────────────────┤
│ [记账] [仪表盘] [刷新]│
└─────────────────────┘
```

### 3.3 功能按钮

| 按钮 | 功能 |
|------|------|
| 记账 | 快速打开支出弹窗 |
| 仪表盘 | 打开完整仪表盘 |
| 刷新 | 重新计算所有余额 |

> 💡 当你发现余额数据不对时，先点击"刷新"重新计算。

---

## 四、状态栏

思源笔记底部的状态栏会显示 **"本月支出: ¥X,XXX.XX"**，让你随时了解当月花销。点击可直接打开仪表盘。

---

## 五、数据分析技巧

### 技巧 1：对比月度开销

打开仪表盘，通过趋势图对比最近几个月的支出。如果某个月突然升高，切换到该月查看分类饼图，找出异常开销来源。

### 技巧 2：追踪分类占比变化

每月查看饼图，观察各分类占比是否合理。比如：
- 餐饮占比超过 40%？可能需要控制外食
- 交通占比突然增大？检查是否有可优化的出行方式

### 技巧 3：使用标签追踪特定项目

给特定项目的支出打上标签（如 `装修`、`旅行`），然后通过嵌入块按标签筛选，汇总项目总花费。详见 [嵌入块](./embed-blocks.md)。

### 技巧 4：关注净资产趋势

侧边栏的"净资产"是 **总资产 + 总负债**（负债为负值）。这个数字的长期趋势反映了你的财务健康状况：
- 持续增长：财务状况在改善
- 持续下降：花超了或负债在增加
